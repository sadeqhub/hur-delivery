**Last Updated: January 7, 2026**

## Introduction

Hur Delivery ("we", "our", or "us") respects your privacy and is committed to protecting your personal data. This privacy policy explains how we collect, use, disclose, and safeguard your information when you use our mobile application (the "App").

Please read this privacy policy carefully. If you do not agree with the terms of this privacy policy, please do not access the App.

## Information We Collect

### Personal Information
When you register and use the App, we may collect the following personal information:

- **Contact Information**: Phone number, name, and store name (for merchants)
- **Account Information**: User ID, account type (merchant, driver, or customer), verification status
- **Profile Information**: Profile photo (optional), business address, store details
- **Financial Information**: Delivery fees, order amounts, wallet balance, transaction history
- **Device Information**: Device model, operating system version, unique device identifiers, FCM tokens for push notifications
- **Communication Data**: WhatsApp messages when you use in-app support or send your location

### Location Data
We collect precise location information to provide core delivery services:

- **Merchant Location**: Store/pickup address for order collection
- **Driver Location**: Real-time location tracking during active deliveries (updated every 3 seconds)
- **Delivery Location**: Customer address for order delivery
- **Customer Location Updates**: When customers share their location via WhatsApp to update delivery address

**Important**: Location data is only collected when:
- You are logged in as a driver with an active order or online status
- You have granted location permissions to the App
- Location tracking stops immediately when you go offline or complete your delivery
- Customer location is only shared when explicitly sent via WhatsApp

### Order and Usage Data
- Order details (items, addresses, prices, notes, vehicle type requirements)
- Order status and history (created, assigned, accepted, on the way, delivered, cancelled, rejected)
- Communication records (in-app notifications, support requests, location update requests)
- App usage statistics, crash reports, and performance data
- Voice recordings (when using voice order feature, processed locally and deleted after transcription)

### WhatsApp Integration Data
- Customer phone numbers for sending location requests
- Location coordinates received via WhatsApp
- Message delivery status and timestamps
- **Note**: We use Wasso API for WhatsApp integration - messages are transmitted securely

## Data Security

We implement industry-standard security measures to protect your personal information:

- **Encryption**: All data transmission is encrypted using SSL/TLS (AES-256 bit encryption)
- **Authentication**: Secure phone-based authentication with OTP verification
- **Access Control**: Role-based permissions ensure users only access appropriate data
- **Database Security**: Row-level security policies and encrypted at rest
- **Regular Security Audits**: Ongoing monitoring for vulnerabilities and penetration testing
- **Secure API**: All API communications are authenticated and rate-limited
- **Data Isolation**: User data is logically separated with strict access controls

However, no method of electronic transmission or storage is 100% secure. While we strive to protect your personal information, we cannot guarantee absolute security.

## Data Retention

We retain your personal information for as long as necessary to:
- Provide our services
- Comply with legal obligations
- Resolve disputes
- Enforce our agreements

**Specific Retention Periods:**
- **Active Accounts**: Data retained while account is active

## Location Services

### Why We Collect Location Data
Location data is essential for:
- Connecting customers with nearby drivers
- Providing accurate delivery time estimates
- Enabling real-time order tracking
- Calculating fair delivery fees based on distance
- Optimizing delivery routes
- Updating delivery location when customer sends new address

### How to Control Location Sharing
- **Android**: Settings > Apps > Hur Delivery > Permissions > Location
  - Choose: "Allow all the time" (for drivers), "Allow only while using the app" (for merchants)
- **iOS**: Settings > Hur Delivery > Location
  - Choose: "Always" (for drivers), "While Using" (for merchants)

**Note**: Disabling location services will prevent drivers from accepting and delivering orders.
