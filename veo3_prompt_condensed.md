# 🎬 Veo 3 Prompt - النسخة المختصرة (بدون نصوص)

## Prompt مباشر لـ Veo 3:

```
Create a 60-90 second animated promotional video in Arabic for delivery drivers. NO TEXT ON SCREEN - visual only with voiceover.

STYLE: Clean 2D flat illustration animation, simple and elegant, smooth transitions, minimal effects. Illustration style like modern digital illustrations - flat colors, clean lines, simple shapes, no 3D effects. Colors: Teal/Cyan primary (#008C95), Blue secondary (#1E40AF), Green success (#10B981), Orange warning (#F59E0B). NO TEXT OVERLAYS - only visual icons, graphics, and animations.

SCENES:
1. Opening: Mobile phone with "Hur" logo (visual only, no text) glowing teal/cyan (#008C95), gradient background from teal (#008C95) to blue (#1E40AF), smooth entrance animation.

2. Problem: Cartoon driver on motorcycle looking tired, searching phone, thought bubbles with question marks (no text), frustrated visual effects, Iraqi city background.

3. Solution: Driver opens Hur app, colorful dashboard appears with icons only (orders icon, map icon, earnings icon), background brightens, driver smiles.

4. Receiving orders: Phone notification appears with notification icon and visual alert (no text), order details shown as icons and graphics, large green "Accept" button (icon only, no text) glows, driver taps with success effect.

5. Navigation: Interactive map with red dot (driver) moving on blue route line, store icon and house icon, direction arrows, GPS voice guidance in Arabic (audio only, no text bubbles).

6. Delivery: Driver reaches customer, camera takes photo with flash, large checkmark ✓ appears (no text), digital wallet appears with balance shown as numbers only (no currency text), numbers increase: 0 → 15000 → 30000, golden coins effect.

7. Features: Split screen showing: offline mode (WiFi icon with X then checkmark), trip history (visual list with icons), animated earnings charts (graphics only), digital wallet (numbers only). Each section zooms in alternately.

8. Success: Happy driver raising hands, bright background, statistics shown as numbers only (50+, 500000+) with icons, glowing star effect.

9. Call to action: Large Hur logo (visual only) rotating with glow, app store icons (Google Play and App Store logos, no text), QR code, gradient background.

MUSIC: Upbeat modern Arabic music, medium-fast tempo, escalates in positive scenes.

SOUND EFFECTS: Alert for new order, success sound for acceptance, GPS voice in Arabic, camera shutter, money adding sound.

ARABIC VOICEOVER ONLY (no on-screen text):
"مرحباً بك في حر - منصة التوصيل الذكية. كنت تبحث عن طلبات توصيل؟ تريد زيادة دخلك بسهولة؟ مع تطبيق حر، كل شيء أصبح سهلاً. استقبل إشعارات فورية بطلبات جديدة - اقبل أو ارفض - أنت تتحكم. ملاحة مباشرة مع خرائط تفاعلية - تتبع دقيق. أرباح فورية تُضاف تلقائياً لمحفظتك. يعمل حتى بدون إنترنت. انضم لآلاف السائقين الناجحين. حمّل تطبيق حر الآن - ابدأ كسب المال اليوم. حر - توصيل سريع، أرباح مباشرة."

VISUAL: Clean 2D illustration style - flat colors, simple shapes, clean lines, minimal shadows. Consistent illustrated driver character, simplified Iraqi city details, smooth motivating movement, prominent brand colors, NO TEXT - only icons, graphics, numbers, and visual elements. Simple illustration effects - no heavy glows, subtle stars, clean animated lines.

TECHNICAL: 1080p or 4K, 16:9 or 9:16, 30-60 FPS, stereo audio.
```
